Pengaturan

Simpan URL Google Apps Script Web App agar data terkirim ke Google Sheets.

Instruksi setup Google Apps Script

  1. Buat Google Sheet baru untuk data MTP MLEB.
  2. Buka Extensions, lalu Apps Script.
  3. Salin isi file apps-script.js ke editor Apps Script dan simpan.
  4. Pilih Deploy, lalu New deployment.
  5. Pilih type Web app.
  6. Set Execute as menjadi Me.
  7. Set Who has access menjadi Anyone.
  8. Klik Deploy, lalu salin Web App URL ke kolom pengaturan ini.
